This project is a clean one-page personal portfolio website designed to showcase essential front-end skills.
It includes:

Responsive navigation bar

Hero section with text + image

About section

Skills section (CSS Grid)

Projects section (Flexbox/Grid)

Contact form

Footer with back-to-top link

It demonstrates the use of Flexbox for alignment, CSS Grid for structuring the skills section, and media queries for a fully responsive design, ensuring smooth layout adjustments across devices.
